# SafeSphere

Women's safety emergency app.

## Folders

### /android
Android app — emergency detection, shake trigger, keyword detection, fake calls.

Open `/android` in Android Studio to build and run.

### /admin
Admin dashboard — Next.js, Supabase, real-time analytics, incident management.

```bash
cd admin
npm install
npm run dev
```

Open http://localhost:3000/admin/login
